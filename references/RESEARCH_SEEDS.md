# 文献与开源项目种子清单

这些是检索起点，不是可以未经核验直接引用的最终清单。Codex 必须阅读摘要/正文或官方项目说明，核对作者、年份、方法和结论后再写入论文。

## 中文研究种子

1. 姜富伟等：《央行货币政策报告文本信息、宏观经济与股票市场》，《金融研究》，2021。  
   https://www.jryj.org.cn/CN/abstract/abstract897.shtml
2. 董青马等：《央行沟通与资产价格——识别“潜在”未预期货币政策信息》，《金融研究》，2024。  
   https://www.jryj.org.cn/CN/abstract/abstract1334.shtml
3. 尚玉皇等：《预期的博弈：央行沟通与国债收益率曲线》，《金融研究》，2025。  
   https://www.jryj.org.cn/CN/abstract/abstract1520.shtml
4. 中国央行预期管理、货币政策沟通、系统性金融风险和资产价格相关的中文权威期刊研究。

## 国际机构与英文研究方向

4. 小样本事件研究、Bootstrap、置换检验和多重比较校正方法文献。

## 筛选标准

- 优先原始论文、官方机构、官方项目页；
- 记录许可证、最后更新、依赖和数据来源；
- 低星项目只能作为思路参考，不能作为方法正确性的唯一证据；
- Medium、博客和二手教程只能用于理解工程流程，不能替代正式文献；
- 不复制大段代码；如确需复用，保留许可证和来源说明。
